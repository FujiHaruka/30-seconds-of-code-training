# 30-seconds-of-code-training

Code training site using snippets in [30-seconds-of-code](https://github.com/30-seconds/30-seconds-of-code). All snippets and tests is from the project.

## Development

Install.

```
$ yarn
```

Build.

```
$ yarn snippets
$ yarn build
```

Deploy to gh-pages.

```
$ yarn deploy
```
